# second_start

Ein einfaches Tetris-Spiel in Python mit `tkinter`.

## Steuerung

- **← / →**: Block nach links / rechts bewegen
- **↓**: Block schneller nach unten bewegen
- **↑**: Block drehen
- **Leertaste**: Block sofort fallen lassen

## Starten

```bash
python3 main.py
```

## Voraussetzungen

- Python 3
- `tkinter` (ist bei den meisten Python-Installationen bereits enthalten)
